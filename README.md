# 🚚 Simulador de Entregas FIFO

## 📌 Descrição
Este projeto simula uma transportadora que realiza entregas utilizando o conceito de Fila (FIFO - First In, First Out).

## ⚙️ Tecnologias
- HTML
- CSS
- JavaScript
- API: https://randomuser.me/

## 🚀 Funcionalidades
- Adicionar pacotes via API
- Exibir fila de entregas
- Entregar pacote (FIFO)
- Histórico de entregas
- Simulação com delay e animação

## 🧠 Conceito aplicado
Fila (FIFO): o primeiro elemento inserido é o primeiro a ser removido.
